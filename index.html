Wat is linux

Linux is simpel gezegd een besturingssyteem. Linux is gratis en is open-source.

Linux kan door iedereen bewerkt worden en mag onder de creative-commons licentie uitgebracht worden.
Zo'n linux versie gemaakt door iemand heet een "distributie".
Een linux distributie of distro is dus een versie van linux gemaakt door een persoon.
